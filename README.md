# Food Delivery App (Backend + Frontend Swift App)

**Delicious Things 2 Home** is a food delivery service for desserts :cake: & biscuits :chocolate_bar: ... yummmmm!. This repo contains the RESTful API.


## How It's Cooked?

* Python 3.6 or higher.
* Django 1.11
* Bootstrap 3.*
